Serapion Movie/Shows

Application developed in laravel8 for back-end, has 2 endpoins describe later. API connects with a local database (mysql), the data connection are in .env. To connect with the api provider uses guzzle and the URL and APIKEY are in .env too. You have to run php artisan migrate and start the server. React for front-end in the same application of laravel you have to run npm run watch
